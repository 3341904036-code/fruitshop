# 水果店管理系统

一个基于Java的应用程序，用于管理水果店的库存、销售和运营。

## 项目概述

这是一个使用Java构建的综合水果店管理系统。该应用程序提供库存跟踪、销售处理和一般店铺操作功能。

## 功能特性

- 库存管理
- 销售跟踪
- 产品目录
- 库存监控
- 报表功能

## 快速开始

### 前置要求

- Java开发工具包(JDK) 8或更高版本
- Apache Maven

### 安装

1. 克隆仓库
2. 导航到项目目录
3. 使用Maven构建项目：
   ```bash
   mvn clean install
   ```

### 运行应用程序

```bash
mvn exec:java -Dexec.mainClass="com.fruitshop.Main"
```

## 项目结构

- `src/main/java/` - 源代码文件
- `src/main/resources/` - 配置和资源文件
- `pom.xml` - Maven配置文件
